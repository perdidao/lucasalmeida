# Boilerplate específico para rodar no Codekit   

## Orientações globais   

Projeto otimizado para Codekit, com as seguintes features:   

- Compass
- JSHint
- JSLint
- Uglify.js

## Plugins inclusos (js/plugin/)

- [API Geográfica](https://bitbucket.org/icomp/api-geogr-fica)
- Owl Carousel
- Fancybox
- HTML5 Shiv
- Masked Input
